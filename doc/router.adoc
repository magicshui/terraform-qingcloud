=== 创建路由器

----
resource "qingcloud_router" "first" {
  name        = "first"
  type        = 1
  description = "this is a test"
  vpc_network = "192.168.0.0/16"
}
----

.参数
[options="header"]
|====
| 名称  | 必选    | 解释
| name  | {y}   | 名称
| type  |   {y} | 路由器类型: 0 - 中型，1 - 小型，2 - 大型，默认为 1
|  description | {n}   | 说明
|vpc_network  |  {n}
|
----
VPC 网络地址范围，目前支持 192.168.0.0/16 或 172.16.0.0/16 。<1>
----
|====

<1> 此参数只在北京3区需要且是必填参数


=== 创建路由器规则

----
resource "qingcloud_router_static" "first" {
  name   = "first"
  router = "${qingcloud_router.first.id}"
  type   = "1"
  val1   = "8080"
  val2   = "192.168.1.2"
  val3   = "8083"
  val4   = "tcp"
}
----

.参数
[options="header"]
|====
| 名称  | 必选    | 解释
| name  | {y}   | 名称
| router | {y} | 路由器 ID
| type | {y} | 类型
| val1 | {y} |
| val2 | {y} |
| val3 | {y} |
| val4 | {n} |
| val5 | {n} |
|====
