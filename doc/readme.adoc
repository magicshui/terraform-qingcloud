为了使用 Terraform QingCloud ，需要在后台拿到 https://console.qingcloud.com/access_keys/[API 密钥]，你需要提供如下信息：

----
provider "qingcloud" {
	id = "yourID" // <1>
	secret = "yourSecret"  // <2>
	zone = "zone"  // <3>
}
----
<1> API 密钥的ID
<2> 密钥值
<3> 可用区，参见可用区表<<zones>>
.可用区表
[[zones]]
