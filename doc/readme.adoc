为了使用 Terraform QingCloud ，需要在后台拿到开发者的 API Key，你需要提供如下信息：

----
variable "id" {
	default = "yourID"
}

variable "secret" {
	default = "yourSecret"
}

variable "zone" {
	default = "pek3a"
}

provider "qingcloud" {
	id = "${var.id}" // <1>
	secret = "${var.secret}"  // <2>
	zone = "${var.zone}"  // <3>
}
----
<1> 密钥的ID
<2> 密钥
<3> 可用区
